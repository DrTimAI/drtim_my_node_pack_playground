This node pack might be interseting to you if:
1. You want to add easy drop down menus to select an LLM within a ComfyUI workflow when using the Mistral API or the Openrouter API.
2. You want to add easy drop down menus to select aspect ratios for use with other nodes where feeding them the aspect ratio is supported but not included with the node.

You can easily edit the strings available in the dropdown menus in the nodes via the associated txt files in the root directory.
